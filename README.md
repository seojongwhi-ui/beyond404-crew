# beyond404-crew

LG SwapIt 수거 크루용 앱 폴더입니다.

이 앱은 사용자가 ThinQ 앱에서 수거 요청을 보냈을 때, 수거 담당자가 요청을 확인하고 수락한 뒤 수거 진행 상태를 업데이트하는 데 사용합니다.

## 기본 역할

```text
1. 수거자 정보 입력
2. 수거 가능 상태로 전환
3. 신규 수거 요청 확인
4. 수거 요청 수락
5. 고객 위치로 이동
6. 가전 수거 완료 처리
```

## 예상 기술 스택

```text
Frontend: Next.js + TypeScript + Tailwind CSS
Backend: beyond404-server와 API 연동
```
